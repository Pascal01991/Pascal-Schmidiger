## ES6 Umbau

Files von Lektion 3 kipiert und auf ES6 umgebaut

export mittels:
'export function sum(a, b) {
return a + b;
}'
oder
'function sum(a, b) {
return a + b;
}

export default sum;'

import mittels:
'import sum from "./sum.js";'

#### Zusammenfassende Schritte:

- Anpassung Syntax für Export/Import
- Installation via npm install babel....
- babel.config.js
